# travel-website

## Problem
Many travelers face difficulty in finding reliable travel information,
packages, and destination details on a single platform. Searching
multiple websites is time-consuming and confusing.

## Solution
This travel website provides a simple and user-friendly platform where
users can explore destinations, view travel packages, and get complete
travel information in one place.

## Features
- Explore popular travel destinations
- View travel packages and details
- Responsive design for mobile and desktop
- Clean and easy-to-use interface

## Tech Used
- HTML
- CSS
- JavaScript

## How to Run
1. Download or clone this repository
2. Open `index.html` in any web browser
3. Start exploring travel destinations
